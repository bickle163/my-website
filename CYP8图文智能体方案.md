# CYP8图文智能体方案

## 1. 核心判断

最适合你的不是单纯的“AI画图工具”，而是一个面向医疗康复B2B获客的图文内容生产智能体。

它应该把以下能力整合到一起：

- 产品知识库：CYP8的技术原理、适用场景、目标客户、价格定位、竞品对比、合规表达边界。
- 素材库：真实产品图、治疗场景图、欧美老人/LTC长护中心场景、品牌Logo、展会照片、医生/护理人员素材。
- 平台风格引擎：Facebook、Google广告、独立站、小红书、朋友圈、LinkedIn等不同平台的版式、语气和尺寸。
- 文案生成器：按目标客户、痛点、卖点、CTA、语种生成标题、正文、图片中文字、广告文案。
- 图像生成与合成：把真实产品图和真实/生成场景结合，输出适合投放的成品图。
- 合规审查：避免夸大疗效、绝对化承诺、违规医疗宣称、暗示平台知道用户健康状况。

## 2. CYP8内容定位

从现有CYP8材料看，适合长期围绕以下主线生产内容：

- 技术主线：wIRA水滤红外A、7-10cm深层能量聚焦、非侵入、精确温控。
- 临床价值主线：慢性疼痛管理、老年康复、术后康复、组织修复环境支持。
- 机构经营主线：适合中小型康复中心、疼痛诊所、养老院、社区护理中心，强调高端技术与中端价格。
- 竞品对比主线：与Hydrosun、冲击波、高能激光等设备做温和的差异化表达。
- 长护中心主线：减少护理负担、丰富康复服务项目、提升住民舒适度和满意度。

## 3. 推荐产品形态

### 形态A：智能体工作台

用户只需要输入：

- 发布平台：Facebook / Google Ads / 独立站 / LinkedIn / 小红书 / 朋友圈
- 内容类型：封面图文 / 广告图文 / 独立站Banner / 轮播图 / 详情页模块 / 短文配图
- 风格：科技感 / 温暖护理感 / 专业医学感 / 展会招商感
- 目标客户：LTC长护中心 / 康复中心 / 疼痛诊所 / 经销商 / 养老院采购负责人
- 主题：慢性疼痛 / 疮面愈合管理 / 老年护理 / 设备招商 / 竞品替代
- 语种：英文 / 波兰语 / 德语 / 法语 / 中文
- 是否使用真实产品图：是/否，选择素材

系统输出：

- 1-3张成品图
- 图片中文字
- 社媒正文
- 广告标题
- CTA
- 备用文案
- 合规风险提示
- 可复制的英文图像生成提示词

### 形态B：批量内容工厂

适合日常运营：

- 输入一个月主题，例如“欧洲LTC慢性疼痛管理获客内容”
- 自动生成30条内容选题
- 每条生成图像方向、标题、正文、CTA、配图建议
- 选中某几条后批量生成图片
- 导出为Excel或内容日历

### 形态C：销售素材生成器

适合业务员、经销商、展会前使用：

- 一键生成客户拜访图文
- 一键生成WhatsApp/Email跟进文案
- 一键生成经销商招商海报
- 一键生成独立站产品模块
- 一键生成不同国家版本

## 4. 关键模块

### 4.1 产品知识库

先把CYP8资料整理成结构化知识：

- 产品一句话定位
- 技术原理
- 适用人群与机构场景
- 不能说的词
- 可以说的合规表达
- 常见客户问题
- 竞品对比
- 证书、研究、临床资料

### 4.2 素材库

建议素材按以下目录管理：

- product_real：真实产品图
- product_cutout：抠图透明背景产品图
- ltc_scene：长护中心场景图
- elderly_people：欧美老人场景图
- clinicians：医生、康复师、护理人员图
- wound_care：疮面护理相关图
- pain_rehab：慢性疼痛康复场景图
- brand：Logo、字体、色彩、品牌规范
- exhibition：展会和工厂真实照片

### 4.3 平台模板

每个平台需要不同策略：

- Facebook：温暖、真实、少术语，强调机构护理场景和住民舒适体验。
- Google Ads：标题短、关键词明确，落地页承接强，避免过度健康个性化表达。
- LinkedIn：专业、B2B、面向机构负责人和采购决策者。
- 独立站：产品真实、证据清晰、CTA明确，适合“预约演示/获取报价/成为经销商”。
- 小红书/朋友圈：更适合中文市场或招商朋友圈，表达可更口语、更故事化。

### 4.4 合规审查

医疗康复设备图文建议避免：

- “治愈”“根治”“保证有效”“永久解决”
- 未证实的临床数据
- 夸张前后对比图
- 直接说“你正在慢性疼痛/你有伤口问题”
- 过度血腥或令人不适的疮面图片
- 暗示替代医生诊疗

建议多用：

- “supports”
- “may help”
- “designed for”
- “non-invasive”
- “for professional rehabilitation settings”
- “supports comfort and recovery management”
- “consult clinical protocols and local regulations”

## 5. 最小可用版本

第一版不需要做很重的软件，可以先做一个本地工作流：

1. 整理CYP8知识库和合规词库。
2. 建立素材库目录。
3. 做一个内容需求表单。
4. 生成结构化输出：文案、图片提示词、版式建议。
5. 接入图像生成或设计工具，输出成品图。
6. 人工最后审核。

第一版的目标是让你输入一句话，例如：

> 做一张Facebook温暖风格封面图，面向欧洲LTC长护中心采购负责人，主题是老年慢性疼痛管理，用真实CYP8产品图加欧美老人护理场景，英文文案。

系统输出：

- 封面图提示词
- 成品图
- 图中文字
- Facebook正文
- CTA
- 合规检查结果
